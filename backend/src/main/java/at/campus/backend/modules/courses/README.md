# Module: Courses

## Responsibility
Manages courses offered by the campus. Courses are the central domain
entity to which other modules are related.

## Core concepts
- Course
- Title
- Description
- ECTS
- Language
- Abbreviation
- Lecturers

## Ownership rules
Courses are global entities.
Only moderators can create, update or delete courses.
